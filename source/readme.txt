# Sauce? What now? #
####################

If you already have your game in ISO+WAV+CUE format:
  You should have 22 tracks. Track02 and Track22 is in ISO format, others should be in WAV. 
  Put the files in the 'iso_wav_cue' folder.

If your game is in BIN+CUE format:
  You should either have one big bin-file or 22 bin-files (one per track).
  Put the files in the 'bin_cue' folder.

The bin-file will be converted to ISO/WAV.
If mutliple bin-files they will be merged before convertion.

The DraculaX.cue file is for after the convertion to ISO/WAVE since bchunk don't create a .cue file.
The Rondo of Blood.cue is used in the final output.
